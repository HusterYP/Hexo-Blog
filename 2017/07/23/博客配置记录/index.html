  • 在提交一篇以前的学习笔记（见C语言指针与数组）时，因为以前是用NotePad写的，现在想要同步到博客上，可是在提交的时候出了问题，（准确的说是在hexo generate的时候报错了），错误为

    Template render error: (unknown path) [Line 31, Column 29] expected variable end
    去Google以后说是出现{-{(此见后面注)的语法错误，于是根据上面的错误提示在对应的行列发现出现了下列代码

    int Array[2][3] = {-{1,2,3},{4,5,6}-};

这里确实出现了{-{，于是果断改为{ {,就可以提交了

注 ： 上面的{-{其实中间没有-两个{是连在一起的，这里是为了能够提交，故意加的，不然还是会报错咯
