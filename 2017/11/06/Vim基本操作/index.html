前言

Vim基本操作总结

  1. u可以撤销上一步操作

  2. VIM的操作模式

    • 命令模式
    • 输入模式
    • 底行模式

  3. vim 文件名

表示用VIM编辑器打开该文件，如果文件不存在，则创建
刚刚打开的时候默认是命令模式，无法输入文件内容，可以输入i切换至输入模式
输入内容后，按esc键进入底行模式，然后 :wq 保存退出
使用vim进入文件内容以后，可以使用dd（连敲）删除首行

  1. VIM的命令格式

    • :q!

      vim进入文件内容后，输入这个也可以退出

    • vim + 文件名

      表示进入文件并将光标定位到最后一行
      也可以指定定位
      vim +3 文件名
      表示光标指定定位到第三行

    • vim +/xxx 文件名

      表示光标定位到首次出现xxx的那一行，然后用n可以切换（如果不止一行出现了xxx的话）

    • vim aa bb cc

      如文件不存在，则可以同时创建多个文件并打开，若存在则同时打开多个文件
      可以输入
      :n 表示在打开的多个文件之中切换
      :N 表示的是回到是一个文件
      :prev 也可以切换到上一个文件

  2. 常用指令

底行模式中

  • :w 表示保存修改
  • :q 退出打开的文件
  • :! 表示强制执行
  • :ls 表示列出当前编辑器中打开的所有文件
  • :3 表示光标快速定位到第三行
  • /xxx 表示光标定位到出现xxx的行(向后搜索)
  • ?xxx 表示从光标位置向前搜索，定位到第一次出现xxx的位置

命令模式中

  • h 光标左移
  • j 光标下移
  • k 光标上移
  • l 光标右移
    　* ctrl + f 向下翻页
  • ctrl + b 向上翻页
  • ctrl + d 向下翻半页
  • ctrl + u 向上翻半页
  • dd 删除光标所在行
  • o 在光标所在行的下方插入一行并切换到输入模式
  • yy 复制光标所在的行
  • p 在光标所在行的下方粘贴（小写）
  • P 在光标所在行的上方粘贴（大写）

  1. 磁盘管理

    • df

    查看磁盘分区的使用状况
    后面可以加
    -l 仅显示本地磁盘（默认）
    -a 显示所有文件系统的磁盘使用情况
    -h 以1024进制计算最合适的单位显示磁盘容量
    -H 以1000进制计算最合适的单位显示磁盘容量
    -T 显示磁盘的分区类型
    -t 显示指定文件系统的磁盘分区，后面还要加参数，如df -l -t ext4
    -x 不显示你是指定类型文件系统的磁盘分区，后面还要加参数，如df -l -t ext4

    • du

    统计磁盘上文件的大小
    -b 以byte为单位统计文件
    -k 以KB为单位统计文件
    -m 以MB为单位统计文件（注意以bkm参数统计出来的文件大小会四舍五入显示）
    -h 按照1024进制以最合适的单位统计文件
    -H 按照1000进制以最合适的单位统计文件
    -s 指定统计目标，后面加文件名

  2. fdisk

    对磁盘分区

    fdisk -l
    列出磁盘分区表

    fdisk 磁盘路径
    表示进入分区模式

  3. 磁盘分区模式

    MBR分区模式

    1. 主分区不超过4个
2. 单个分区容量最大为2TB

    GPT分区模式

    1. 主分区个数“几乎”没有限制
2. 单个分区容量“几乎“没有限制
